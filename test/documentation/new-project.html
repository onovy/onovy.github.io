Nový projekt

Vytvoření nového projektu je s Jekyllem velmi jednoduché.

Úprava _config.yml

Hlavní konfigurační soubor, viz. YAML.

{% highlight bash %} # Seznam projektů projects: # Testovací projekt # Klíč projektu - key: Testovací # Title title: Seznam opensource # Cesta k index.html url: /test/index.html # Popis projektu (zobrazí se na HP a v detailu projektu) description: > Projekt, na kterém se snažíme ukázat práci s Jekyllem, vytvoření nového projektu do služby Opensource Seznam. # Klíčová slova keywords: > opensource,showcase,project # Informace o verzi version: 1.2 # Odkaz ke stažení download: https://github.com/seznam/seznam.github.io.git # Výhody použití (zobrazí se v detailu projektu) features: - title: Jekyll perex: > Transformuje plain text na statické webové stránky a blog. - title: Pygments highlighter perex: > Zvýrazňovač syntaxí v Pythonu s podporou 100 jazyků. - title: Pre-defined layouts perex: > Předchystané šablony pro jednotlivé části projektů. # Navigační lišta menu: - link: title: "About" url: "/test/index.html" - link: title: "Začínáme" url: "/test/documentation/index.html" - link: title: "Vytvoření nového projektu" url: "/test/documentation/new-project.html" {% endhighlight %}

Adresář projektu

V rootu je nutné vytvořit adresář se stejným názvem jako je key v _config.yml.

HTML šablony

ROOT/klic_projektu/index.html

Kromě nastavení se zde píše demo, které se zobrazuje v detailu projektu na úvodní stránce.

{% highlight html %} --- # Výběr šablony layout: detail # Klíč projektu (stejný jako v _config.yml) projectkey: test # Title (přetíží ten v _config.yml) title: Seznam Opensource showcase ---

Demo

:-)
{% endhighlight %}

ROOT/klic_projektu/documentation/index.html

Pokud bude url v definici panelu odkazovat na kotvu, tak se menu bude při scrollování v obsahu automaticky označovat.
HTML se píše do jednotlivých sekcí, strukturovaně pod sebe.

{% highlight html %} --- # Výběr šablony layout: documentation # Klíč projektu (stejný jako v _config.yml) projectkey: test # Title (přetíží ten v _config.yml) title: Test docs page # Definice levého panelu panel: - link: title: Clone project url: "#clone" - link: title: Jekyll url: "#jekyll" ---
...
..
{% endhighlight %}
