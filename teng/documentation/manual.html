Introduction

Teng (template engine) is general purpose template tool. Primarily a C++ library but also available as a Python module. It composes web pages, mail messages or any text files by combining input templates with data supplied by an application and optional configuration and localization dictionary.

Characteristics

  • Separation of application code and design templates
  • Multiple templates
  • Dictionaries - language dependent or not
  • Caching of templates and dictionaries - automatic update on change
  • Possible data integrity control
  • API for C++ and Python
  • Much more :-)
